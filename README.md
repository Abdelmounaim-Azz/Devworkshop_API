# Devworkshop_API
Nodejs Back End API for devworkshop website using Express and MongoDB

> Backend API for devworkshopr application.

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
